# Browser Game 3D Asset Pack v1

This pack contains **50 real `.glb` 3D assets** for a stylized 3D browser game.

## Included Categories

Industrial/corporate props, forge hazards, forest sanctuary objects, orbital/cosmic set pieces, resonant sound-combat props, urban transit/tech objects, data vault/archive objects, pickups, telegraphs, modular arena pieces, weapons, common hazards, and destructibles.

## Engine Notes

These are static low-poly GLB meshes. They can be loaded in Three.js, Babylon.js, PlayCanvas, Godot, Unity, or Blender. They are not animated, rigged, scripted, or converted into prefabs. Treat `colliders.json` as a first-pass collider guide.

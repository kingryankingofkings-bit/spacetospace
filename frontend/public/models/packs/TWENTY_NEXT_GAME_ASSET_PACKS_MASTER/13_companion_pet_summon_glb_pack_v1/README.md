# Companion Pet + Summon GLB Pack v1

Summon circles, pet nests, drone stations, and companion interaction props.

## Contents

This pack contains **12 real `.glb` assets**.

## Files

- `glb_assets/` — actual GLB meshes
- `manifest.json` — asset metadata
- `colliders.json` — suggested collider/bounds data
- `ASSET_INDEX.md` — human-readable asset index
- `VALIDATION_REPORT.json` — load validation
- `threejs_loader_example.js` — simple GLB loading helper

## Production Notes

These are static low-poly prototype meshes. They are suitable for browser-game blockout, gameplay testing, world dressing, and asset placeholders. They are not rigged, skinned, animated, scripted, or engine-prefabbed.

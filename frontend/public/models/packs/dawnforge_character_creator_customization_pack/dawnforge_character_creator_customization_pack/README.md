# Dawnforge Character Creator Customization Pack

This pack contains a detailed character creation and customization process outline for Echoes of the Dawnforge. It includes deep identity, body, face, hair, skin, scars, tattoos, prosthetics, accessories, class imprint, voice, animation, starter gear, preview, technical implementation, and QA coverage.

Files:

- `CHARACTER_CUSTOMIZATION_PROCESS_OUTLINE.md` — main production outline.
- `CHARACTER_CUSTOMIZATION_PROCESS_OUTLINE.txt` — plain-text copy.
- `QUICK_REFERENCE_SELECTION_COUNTS.md` — compact count summary.
- `data/customization_catalog.json` — structured catalog for tool import or future expansion.
- `validation_report.txt` — confirms requested minimums.

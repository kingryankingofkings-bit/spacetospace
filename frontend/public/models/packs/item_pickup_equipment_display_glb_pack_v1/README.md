# Item Pickup + Equipment Display GLB Pack v1

This pack contains **75 real `.glb` assets** for item pickups, equipment displays, rarity visuals, loot chests, reward pedestals, and shop display stands.

## Why this pack exists

The game can have thousands of item records without needing thousands of handcrafted models. These assets act as reusable visual archetypes:

- sword, axe, mace, spear, bow, staff, shield, armor, accessory, currency, crafting material
- rarity rings and loot beams
- loot chests and boss reward chests
- shop display stands and rotating item bases

## Suggested use

Map your item database to these model archetypes by item type and rarity. Use engine-side material overrides, particles, labels, and stat-card UI to create variety.

## Files

- `glb_assets/` — actual GLB assets
- `manifest.json` — metadata and tags
- `colliders.json` — recommended collider setup
- `ASSET_INDEX.md` — human-readable index
- `VALIDATION_REPORT.json` — load validation
- `threejs_loader_example.js` — simple loader example

# Dawnforge Sky Wardens Epochalypse Postgame Pack

This is the updated Sky Wardens postgame pack with the 100-quest secondary storyline rewritten around **The Order of the Epochalypse** as the shadow organization villains.

## Contents preserved from the prior Sky Wardens pack

```text
models/characters/       50 static low-poly Sky Warden NPC .glb files
models/skyships/         12 static low-poly skyship .glb files
models/headquarters/     1 static low-poly Sky Wardens headquarters .glb file
docs/                    roster, rank, HQ, ship, asset, reward, quest, and villain documents
storyline/               rewritten 100-quest postgame chain
rewards/                 100 unique Godly rewards, exactly 10 per class
tables/                  CSV import tables
data/                    JSON import data
validation/              validation report
```

## Updated villain requirement

The postgame villains are **The Order of the Epochalypse**, spelled exactly that way. The Order is a shadow organization that tries to edit, curate, and overwrite eras of history after the final boss is defeated.

## Torinn Rhogar rule

Every one of the 100 postgame quests is given by Torinn Rhogar and turned in to Torinn Rhogar. Torinn Rhogar is **not** included as a character model and is not added to the roster.

## Quest and reward rules

- 100 postgame quests.
- 3–5 required subquests per quest.
- 0 XP from every quest.
- 1 unique Godly reward per quest.
- 100 total Godly rewards.
- 10 Godly rewards per playable class.
- Existing GLB assets, Sky Warden roster, skyships, and headquarters are preserved.

## Important production note

The `.glb` files are valid static low-poly/prototype assets for blockout, pipeline testing, and implementation placeholders. They are not final rigged AAA character sculpts.

# VFX Proxy + Combat Telegraph GLB Pack v1

This pack contains **68 real `.glb` mesh assets** for combat readability and visual effects prototyping.

## Included VFX Types

- Slash arcs and melee trails
- Hit impacts, block sparks, parry flashes
- Fire, ice, lightning, poison, acid effects
- Root, nature, healing, shield, and buff effects
- Void, gravity, phase-shift, and zero-G effects
- Sonic/rhythm attack proxies
- Beam attacks and projectiles
- Boss phase-change effects
- Warning circles, cones, lines, rectangles, and projectile indicators
- Combo finisher, aerial juggle, vault-kick, and grapple trail proxies

## Engine Notes

These are static mesh proxies. For final production, your engine should add emissive materials, additive blending, timed spawn/despawn, scaling curves, billboarding where useful, particle systems, and synchronized audio cues.

## Files

- `glb_assets/` — actual GLB assets
- `manifest.json` — asset metadata
- `colliders.json` — suggested bounds/collider information
- `ASSET_INDEX.md` — human-readable index
- `VALIDATION_REPORT.json` — load validation
- `threejs_loader_example.js` — simple browser loading example

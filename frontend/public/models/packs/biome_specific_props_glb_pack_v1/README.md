# Biome-Specific Props GLB Pack v1

This pack contains **72 real `.glb` assets** across **12 biomes**.

## Biomes Covered

- cave
- city
- cosmic_data
- desert
- forest
- industrial
- ruins
- skyship
- snow_ice
- swamp
- underwater
- volcanic

## Purpose

Use this pack to make the 341 game areas feel visually distinct. These assets are intended for fast placement, instancing, and browser-friendly world dressing.

## Files

- `glb_assets/` — all GLB files
- `manifest.json` — asset metadata
- `colliders.json` — suggested collider data
- `ASSET_INDEX.md` — human-readable index
- `VALIDATION_REPORT.json` — load validation
- `threejs_loader_example.js` — simple loader example

## Notes

These are static low-poly mesh assets. They are not animated, rigged, scripted, or configured as engine prefabs.

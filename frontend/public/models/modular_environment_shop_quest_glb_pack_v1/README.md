# Modular Environment + Shop + Quest GLB Pack v1

This pack contains **57 real `.glb` assets** covering three major needs:

1. Modular environment pieces
2. Shop / town / inn / blacksmith interior assets
3. Quest and interaction objects

## Included

- Modular floors, walls, doors, gates, windows
- Stairs, ramps, bridges, ladders, platforms
- Roofs, fences, barricades, terrain chunks, cave/ruin pieces
- Water, lava, and acid surface modules
- Counters, shelves, weapon racks, armor stands, forge props
- Beds, tables, stools, lanterns, banners, market stalls, chests
- Notice boards, pedestals, relics, cages, beacons, switches, sealed doors, pressure plates

## Usage

- Load with Three.js, Babylon.js, PlayCanvas, Godot, Blender, or Unity.
- Use instancing for repeatable modules like walls, floors, fences, chairs, and signs.
- Treat the quest objects as interaction anchors or objective markers.
- Refer to `colliders.json` for starting collider sizes and types.

## Files

- `glb_assets/` — the actual GLB files
- `manifest.json` — all asset metadata
- `colliders.json` — recommended collider setup
- `ASSET_INDEX.md` — human-readable index
- `VALIDATION_REPORT.json` — simple load validation report
- `threejs_loader_example.js` — basic browser loading example

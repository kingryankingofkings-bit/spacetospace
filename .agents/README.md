# Game Engine Asset Integration Master Skill

Install path inside this package:

```text
.agents/skills/game-engine-asset-integration-master/SKILL.md
```

Copy the `.agents` folder into the root of the target Antigravity project/workspace, or copy the `game-engine-asset-integration-master` folder into the project’s existing `.agents/skills/` folder.

This package contains one master skill only. It is instructions-only and includes no executable scripts.

## What it covers

- `.glb` and `.gltf` validation, import, material mapping, animation mapping, runtime import, and runtime generation.
- Other engine asset files: FBX, USD, OBJ, Blender scenes, textures, materials, shaders, audio, VFX, UI, fonts, localization, gameplay data, video, prefabs, Blueprints, PackedScenes, bundles, DLC, live content, and user-generated content.
- Engine adapters for Unity, Unreal Engine, Godot, and custom engines.
- Runtime catalog, asset IDs, dependency graphs, loading, spawning, procedural assembly, multiplayer content-version handling, and card-game-specific content generation.
